DOM Project

For the next portion of the SEP11 course, my class learned about DOM. DOM (Document Object Model) is a programming interface that shows the structure of the website such as content and style. It's also used to manipulate existing websites. By learning how to change text elements and appearance of a website, my class was able to create a project using those skills. So, me and my partner decided to make a random color generator.

It's pretty self-explanatory of what it does, you just click the button and it generates random color. We have three buttons: first button is the main color generator that our teacher assigned us, the second was an extension of adding more random colors, and the third one is a random Kanye West generator since my partner is obsessed with him.

The hardest part of this project was implementing the Kanye West generator because it was messing up our other two generator. Initially, me and my partner wanted one button to randomly generate colors and pictures of Kanye but it was conflicting our code and overall breaking our program so we made three buttons instead.
