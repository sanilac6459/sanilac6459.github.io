FinTech Focus: FTF Weather App

The summer of 2023, I participated in Giant Machines FinTech Focus Program. It's a program for incoming college freshmen that's interested in the tech and finance field.

For this project, me and my group members created a weather API app with Python, Flask which a Python library, and two APIs.

One of the APIs was the Openweathermap API that basically displays the weather of a certain location such as the current weather and the weather in the next three days. The other API was the Geocode API. This helps turns coordinates of the location into the names. It's more convenient to ask the user to input the name of the location rather than the location.

The most challenging part of this project was getting our API to work because there were times we had a bunch of errors and we didn't know how to resolve it. We also had 3-4 days to build this project so managing our time was tough on us.

Overall, I'm proud of how it turned out and the amount of effort me and my group put into it when we had a small period of time to develop this creation.
