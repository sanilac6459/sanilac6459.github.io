APCSA Freedom Project: Discord Music Bot

For this year long project, my class was able to create anything with whatever tool, whether it was Java (the course of APCSA) or not. So, my group and I used Discord Python API which is an API wrapper in Discord that allows developers to create bots or any applications for Discord channels or servers. The programming language of our tool is Python which is used for web and software development.

For our creation, we used our tool to create a Discord Music bot for users to listen to music while they're on a call with their friends. It's a great way to build connections with others and overall build a community and culture in this social and gaming platform.

The most challenging part of creating our project was implementing the main functionality of our bot which was to get it to play, pause, resume, and stop the music. We kept facing numerous errors and we were stuck on this portion of our project for a long time since we didn't know how to fix it. Additionally, my IDE or the place I coded this creation was broken so I has to delete and make a new one which almost made me lose all of my group's code. We also had trouble with time management since all three of us were so busy due to college process so we had to rush and complete it the last minute.

My takeaway was processing my group's code for the project. There were mutliple times where I looked back at our code and my brain would be fried because I didn't understand what our product is able to do so far. It took long hours of reading documentations, watching tutorials, and asking my peers to figure it out.
