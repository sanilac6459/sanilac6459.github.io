Javascript Project

As a way to sum up the basic fundamentals of Javascript, my class was instructed to create a project to demonstrate our understanding of each element such as variables, arrays, loops, functions, and etc. So, my partner and I decided to create a number generator.

Basically, the computer would generate a random number between 1-10 and would prompt the user to guess a number in between those range. If the number they guess is too high or too low, they will be asked to guess again and it continues until they guess the correct number. Once that happens, they get a huge congratulation message on their screen.

The challenging part of this project was getting the user to guess again if their input was incorrect. For some reason, the prompt wasn't reappearing after the user inputs their first guess. After some time, me and my partner figured it out by reusing the code we implemented to make the prompt appear on the screen.
